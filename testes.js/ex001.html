Inicando JavaScript

A tag ( = ) sempre será para RECEBER

Tag (NULL) p/anular algo.

Palavra (VAR ou LET) p/criar Identificadores (um local p/armazenar), sempre numerar eles.

Podem também ser colocadas quantidades a ser recebidas
EX:

var n1 = 5
var n2 = 8.5
var n3 = 15

Podemos também criar Identificadores maiores usando em vez de números, usando letras.
EX:

var s1 = "JavaScript"
var s2 = 'Curso em Vídeo'
var s3 = `Guanabra`

Onde o tipo de aspas usado faz diferença

  • Não podem começar por números.
  • Podem usar acentos.
  • Não podem ter espaços.
  • Não podem ser palavras que fazem parte do sistema
  • Podem começar com Letras, $ ou _.

Temos diferentes tipos de Identificadores, sendo eles.

  • Number, apenas dados de números.
  • String, palavras em geral, mas também podendo ser CPF, Telefone, Número da identidade, etc.
  • Boolean, true ou false.

Typrof - p/saber a funçao daquele Identificador

A tag (PROMPT), recebe sempre o String. Se for um número tem q ser mudado manualmente.

Usando a conversão da tag (Number.parseInt - p/número inteiro) e (Number.parseFloat - p/número com vírgula)

Mas também podemos usar só a tag (Number)

E p/mudarmos de Número para String, usamos a tag (String(n)) ou (n.toString( ))

(Number + Number) = Adição.
(String + String) = Concatenação.

Para conectarmos a frase com o Identificador usado ele não pode estar dentro dos parenteses juntos.

Podemos usar a tag ( ${s} para facilitar a escrita).

Temos outras formas da Strings.

  • s.lenght - para saber quantos caracteres ela tem.
  • s.toUpperCase() - tudo vai para letras Maiúsculas.
  • s.toLowerCase() - tudo para letras minúsculas.

A tag (toFixed (2) ), entre parênteses é a quantidade de casas adicionais no número, so funciona para números.

Se quisermos que mostre em tabela de salário basta usarmos a tag - toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'}) - no lugar BRL botamos a sigla da moeda que queremos.
